
# Bonhomia Joyas

E-commerce for jewelry

# Build

<pre>
        # install dependencies
        *npm install*

        # serve with hot reload at localhost:8080
        *npm start*

        # build for production with minification
        *npm run build*

        # build for production and view the bundle analyzer report
        *npm run build --report*
</pre>


# Tools implemented

* React js
* React Router DOM
* Hooks
* Boostrap
* Formik
* React icons
* Firebase
